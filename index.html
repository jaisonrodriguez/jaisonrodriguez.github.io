API Backend & Frontend CRM

Viabot (Ecosistema)

Plataforma B2B para agencias de viajes. Cuenta con una API altamente segura y un CRM frontend especializado, diseñado para máxima accesibilidad por parte de agentes de la tercera edad.

  • Zero Trust & Fail-Fast: Arquitectura estricta de validación por API Keys y tenant_id (Supabase). Manejo de colas con BullMQ y caché ultrarrápida con Redis.
  • Integraciones B2B: Conexión nativa con Sabre Gateway (reservas aéreas globales) y delegación híbrida de WhatsApp (Bots + Human) vía Puppeteer.
  • Accesibilidad Extrema: Interfaz Next.js adaptada con dictado de notas por voz, bóveda interna para renderizar documentos sin descargas, e ingreso sin contraseñas (OAuth).
Next.js 14 / Express.js TypeScript Redis & BullMQ Sabre API
